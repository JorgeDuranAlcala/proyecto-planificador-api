## Correr en local

Clonar el proyecto

```bash
  git clone https://github.com/JorgeDuranAlcala/proyecto-planificador-api
```

Ir al directorio del proyecto

```bash
  cd proyecto-planificador-api
```


Instalacion de las dependencias

```bash
  npm install
```

Iniciar el proyecto

```bash
# desarrollo
$ pnpm run start

# watch mode
$ pnpm run start:dev

# produccion
$ pnpm run start:prod
```
